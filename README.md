# PostgreSQL Learnings

A structured, hands-on PostgreSQL learning repository.  
All files are self-contained `.sql` scripts with inline comments — designed to be readable after a long gap.

## Topics Covered

- Database & Table Management (DDL)
- Data Insertion, Update, Delete (DML)
- Filtering, Sorting, Aggregations
- Joins (INNER, LEFT, RIGHT, FULL)
- Subqueries & CTEs
- Recursive CTEs
- Indexes & Query Optimization
- Transactions & Constraints
